Comandos git

git status - verifica status

git | add . (adiciona toda a pasta) | "nome-do-arquivo"

git commit -m "frase" - adicionar etiqueta à mudança

git remote - setar endereços de repositório remotos

git push | git push origin main - sobe arquivo

git pull
