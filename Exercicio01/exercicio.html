Como utilizar o CSS externamente

1 - Importar da folha de estilo CSS (stylesheet)

A declaração <link rel="stylesheet" href="./css/imagem.css"> declara a folha de estilo css de uma página onde "rel" é a definição do tipo de arquivo que será importado (neste caso, css) e "href" aponta o diretório onde arquivo se localiza.

2 - Definir de classes CSS

No arquivo .css são declaradas as formas de estilo para serem aplicadas aos elementos HTML, e podem ser feitas de três maneiras, respectivamente, através de 1 - classes (com um "." no começo da declaração);
2 - tags html (tags html em si, como img, p), e o estilo será aplicado para tudo o que for img;
3 - identificações únicas (através de #);

3 - Atribuir estilos aos elementos HTML

Neste exemplo, aplicaremos estilo à uma imagem específica.

No primeiro exemplo de uso, a classe .dimensao foi utilizada para a imagem sob o atributo class.

No segundo exemplo de uso, o estilo é aplicado via definição (conector) img no arquivo .css.

No terceiro exemplo de uso, a aplicação do estilo é determinada pelo uso do id como atributo da tag img.
